# 📜 Random Quote Generator — 1000+ Inspiring Quotes

A beautifully crafted, single‑page web application that instantly serves fresh, thought‑provoking quotes from a pool of over **1000 hand‑picked sayings**. Filter by themes like motivation, wisdom, love, stoicism, creativity and more. Every click brings a new gem – never the same one twice until you’ve explored them all.

## ✨ Features

- **1000+ Unique Quotes** – drawn from philosophers, writers, scientists, entrepreneurs, artists, and modern thinkers.
- **10 Curated Topics** – filter by theme: Motivation, Wisdom, Life, Love, Success, Resilience, Happiness, Leadership, Creativity, Stoicism.
- **Smart Rotation** – never repeats a quote until the active pool is exhausted, then reshuffles.
- **Contextual Insights** – about 30% of quotes include a short, insightful explanation or anecdote.
- **One‑Click Actions** – Copy to clipboard, or listen to the quote spoken aloud (Speech Synthesis).
- **Keyboard Shortcuts** – Press `Space` or `N` for a new quote, `C` to copy.
- **Elegant Dark Interface** – soft gradients, floating particles, and smooth animations.
- **Fully Responsive** – looks great on desktop, tablet, and mobile.

## 🛠 How to Use

1. **Clone or download** the repository.
2. Open `index.html` in any modern browser.
3. Click **“New Quote”** (or press `Space`) to get a fresh random quote.
4. Use the **topic pills** to narrow down quotes to a specific theme.
5. Use **Copy** to grab the quote text, or **Speak** to hear it aloud.

No dependencies, no build steps – it’s a standalone HTML file.
